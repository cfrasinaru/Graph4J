Instead of downloading the .jar manually, you may add the following dependency to the pom.xml of a Maven project: 

	  <dependency>
		<groupId>org.graph4j</groupId>
		<artifactId>graph4j</artifactId>
		<version>1.0.7</version>                          
	  </dependency>

Graph4J requires Java 11 or higher.
Graph4J is a work in progress, there is no guarantee that the latest release is backward compatible with the older ones.
